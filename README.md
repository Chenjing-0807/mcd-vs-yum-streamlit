# 🍔 Fast Food Wars: Financial Risk & Return Analysis (MCD vs. YUM)

## 📌 Project Overview
This project is an interactive financial data dashboard built with **Python & Streamlit**. It evaluates the historical price performance and tail-risk volatility of two fast-food giants: **McDonald's (MCD)** and **Yum! Brands (YUM)**. 

Using market data extracted from the **CRSP academic database**, this tool visualizes cumulative returns and daily return distributions to help investors assess risk-adjusted performance.

## 🛠️ Tech Stack
* **Language:** Python
* **Web Framework:** Streamlit
* **Data Processing:** Pandas
* **Data Visualization:** Matplotlib (with dynamic LaTeX annotations)

## 📊 Key Insights
1. **Return Analysis:** Standardizing the initial investment to $100, MCD demonstrated superior long-term growth and quicker recovery post-2022 market volatility compared to YUM.
2. **Tail Risk Analysis:** The daily return distribution histogram reveals that YUM suffers from a "fatter left tail," indicating a higher historical probability of sharp single-day declines.

---
*Data Source: Center for Research in Security Prices (CRSP) - Daily Stock File*
